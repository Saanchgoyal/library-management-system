
# Library Management System (Python)

A simple file-based library management system.

## Features
- Add a new book  
- Issue a book  
- Return a book  
- Display available books  
- Uses a text file to store book records  

## Python Function used 
<ul>
    <li>File Input Output</li>
    <li>Operating System (OS)</li>
</ul> 

## How to Run
```
python library.py
```

A `books.txt` file will be created automatically when you add books.

## Project Structure
```
library.py
README.md
books.txt (auto-created)
```
